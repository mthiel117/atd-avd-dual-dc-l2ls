## ATD Lab Updates

Update AVD to latest version

``` bash
ansible-galaxy collection install arista.avd --force
```

Install Deepmerge module

``` bash
pip install deepmerge
```

## Initialize WAN environment

Push pre-defined configs for WAN Nodes

``` bash
make prepwan
```
